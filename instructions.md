# COMPILING:
 - To compile, type the following from this folder:
  javac -cp .:bcprov-jdk15on-162.jar Homework2.java

# RUNNING:
 - To run, type the following from this folder:
  java -cp .:bcprov-jdk15on-162.jar  Homework2

  - The program first asks you to input a text. It then encrypts/decrypts the text and prints result of encryption and decryption.
  - When the basic encryption/decryption is done, the program asks if you want to run time comparisons.
  - Entering 'y' runs them and outputs the requested info from the hw2 specification.
  - Entering anything else exits the program.
